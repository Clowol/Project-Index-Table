# Communication Basics_通信方式基础

> 本章将介绍一些常见的**通信方式**和**协议**，帮助你理解不同设备之间如何进行**数据交换**。
> 因为通信方式和协议非常多，在我项目中初始，作为新手都不熟悉，因此犯了很多很基础的错误，后面在师兄的帮助下才逐渐理解了这些通信方式和协议的基本概念和使用方法。希望通过这章的介绍，能够帮助你更好地理解和使用这些通信方式和协议。
> 在深入具体的通信协议（UART、I2C、SPI、CAN 等）之前，有必要先了解几个最基础的概念：**单工、半双工、全双工**，以及**同步/异步**、**串行/并行**、**差分/单端**等。

---

## 1. 串行通信 (Serial Communication):&emsp;简单高效，适合长距离传输

### 1.1 定义
串行通信&emsp;是一种数据传输方式，其中数据以位的形式逐个传输,是一种将数据以二进制代码的形式排成一行，一个接一个地在一条信道上进行传输的方式。

### 1.2 分类方式&emsp;——&emsp;传输方式
> 这三个概念描述的是通信双方**数据流动的方向**。

- **单工 (Simplex)**: &emsp;数据只能**单向传输**。为保证正确传送数据信号，接收端要对接收的数据进行校验，若校验出错，则通过监控信道发送请求重发的信号。<span style="font-size:14px;opacity:0.8;">
例如，电视广播就是单工通信，电视台发送信号，观众只能接收。
</span>

- **半双工 (Half-Duplex)**: &emsp;数据可以**双向传输**，但不能**同时**进行。因此，半双工通信实际上是一种可切换方向的单工通信.<span style="font-size:14px;opacity:0.8;">
例如，对讲机就是半双工通信，双方可以交替说话，但不能同时说话。
</span>

> 举个简单例子，一条窄窄的马路，同时只能有一辆车通过，当目前有两量车对开，这种情况下就只能一辆先过，等到头儿后另一辆再开，因此这种通信方式就叫半双工通信。

- **全双工 (Full-Duplex)**: &emsp;数据可以同时双向传输。全双工通信是两个单工通信方式的结合，要求收发双方都有独立的接收和发送能力。<span style="font-size:14px;opacity:0.8;">
例如，电话就是全双工通信，双方可以同时说话和听。
</span>

### 1.3 分类方式&emsp;——&emsp;数据同步方式
> 这两个概念描述的是通信双方**数据传输的时序**。

- **同步通信 (Synchronous Communication)**:&emsp; 发送和接收双方使用**同步时钟**来协调数据传输。数据以固定的时间间隔发送，接收方根据时钟信号来正确接收数据。<span style="font-size:14px;opacity:0.8;">
例如，I2C 和 SPI 就是同步通信协议。
</span>

- **异步通信 (Asynchronous Communication)**:&emsp; 发送和接收双方不使用同步时钟，而是通过**起始位**和**停止位**来标识数据的开始和结束。数据可以在任意时间发送，接收方根据起始位和停止位来正确接收数据。<span style="font-size:14px;opacity:0.8;">
例如，UART 就是异步通信协议。
</span>

## 2. 并行通信 (Parallel Communication):&emsp;高速传输，适合短距离连接

### 2.1 定义
**并行通信**&emsp;是一种数据传输方式，其中数据以**多个位同时传输**，每个位通过独立的信道进行传输。
并行通信通常用于需要**高速数据传输**的场景，但由于需要更多的线缆和更复杂的电路设计，通常只适用于短距离连接。

### 2.2 分类方式&emsp;——&emsp;信号类型

- **单端通信 (Single-Ended Communication)**:&emsp;每个信号线相对于一个公共地线进行传输。单端通信容易受到电磁干扰，信号质量较差，适用于短距离通信。<span style="font-size:14px;opacity:0.8;">
例如，传统的并行接口（如 Centronics）就是单端通信。
</span>

- **差分通信 (Differential Communication)**:&emsp;每个信号线都有一个对应的反相信号线，信号通过两条线的电压差来传输。差分通信具有更好的抗干扰能力，适用于长距离通信。<span style="font-size:14px;opacity:0.8;">
例如，USB 和 LVDS 就是差分通信协议。
</span>
